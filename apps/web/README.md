## **🚀 애플리케이션 소개**

> ‘왁크래프트’는 170만 유튜버 우왁굳의 마인크래프트 컨텐츠 중 '눕프로해커' 시리즈를 정리한 웹사이트입니다.  
> 방송 중 “이 건축가가 어떤 작품을 만들었지?” 하고 궁금해하는 팬들을 위해, 참여자별로 콘텐츠를 분류하고 검색할 수 있도록 했습니다.
